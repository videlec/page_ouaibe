Programmation (et maths expérimentales)

Je travaille sur des algorithmes autour de la dynamique symbolique, la combinatoire et la géométrie des surfaces.

Je fais partie du projet européen OpenDreamKit (Septembre 2015-Septembre 2019) dont l'objectif est de renforcer les interactions entre les logiciels libres de mathématiques.

Je développe les librairies suivantes

  • pplpy: un wrapper Python à Parma Polyhedra Library (PPL). Le code source est disponible sur github.
  • pypolymake: un wrapper Python à polymake. Le code source est disponible sur github.

Je contribue assez régulièrement au logiciel Sage (pour avoir un aperçu de mon travail vous pouvez consulter les tickets auxquels j'ai participé). J'ai également participé à d'autres projets libres comme comme PARI/GP, MPIR et flint.

Quelques calculs

Spectre de pseudo-Anosov calculé à partir des diagrammes de Rauzy

Ci-dessous vous trouverez des listes d'homéomorphismes pseudo-Anosov obtenus via l'induction de Rauzy dans les strates H(2) et H(1,1). On y trouve les polyonmes caractéristiques de l'action dans l'homologie.

  • H_hyp(2)
    • le diagramme de rauzy
    • liste des spectres
  • H_hyp(1,1)
    • le diagramme de Rauzy
    • liste des spectres
A partir de ces données, on peut étudier les difféomorphismes pseudo-Anosov pour lesquels l'action dans l'homologie a deux valeurs propres 1 (autrement dit cette matrice preserve un sous-espace de dimension 2). Les premiers exemples dans la strate H(2), partant de la permutation a b c d/d c b a, sont:
  • tttbtbtbtb : longueur 10, dilatation 3/2*sqrt(5) + 7/2 ~ 6.854,
  • ttbtbttbttb : longueur 11, dilatation sqrt(15) + 4 ~ 7.873,
  • ttbtbttbtttb : longueur 12, dilatation 1/2*sqrt(77) + 9/2 ~ 8.887,
  • tttbtbtttbtb : longueur 12, dilatation 3/2*sqrt(13) + 11/2 ~ 10.908,
  • tttbtbbtbttb : longueur 12, dilatation sqrt(35) + 6 ~ 11.916.

Quelques liens pour le logiciel Sage
